一、HTML(Hyper Text Markup Language)

    超文本标记语言，标准通用标记语言下的一个应用。

   “超文本”就是指页面内可以包含图片、链接，甚至音乐、程序等非文字元素。

    超文本标记语言的结构包括“头”部分（英语：Head）、和“主体”部分（英语：Body）

         其中“头”部提供关于网页的信息;

        “主体”部分提供网页的具体内容。

    1、HTML是一门解释性语言。

       HTML不需要编译，由浏览器解释执行。


    2、标记的分类：

       1）单标记

          <标签名称 属性="值" ...... />

       2）双标记

          <标签名称 属性="值" ...... >标签体</标签名称>

三、HTML文档流

    什么叫文档流？

        简单来说，就是元素在页面出现的先后顺序。

        一般情况下，HTML按照标签的书写顺序。


    那什么叫“正常文档流”呢？

       正常文档流，将页面自上而下分成一行一行。

           块元素独占一行；

           相邻的行内元素在每行中按从左到右地依次排列元素。


四、HTML元素的分类：

    http://www.gezila.com/tutorials/52292.html

    1）级块元素

       在网页单独占一行。

       边界、填充样式，水平和垂直方向均有效。


    2）行内元素

       多个元素可以在同一行中出现。

       一行中可以存在多个行内元素，从左向右依次排列。

       边界、填充样式，水平方向有效，垂直方向无效。


    3）可变元素

       可以是级块元素，也可以是行内元素。

       同一行中可以多个元素，边界、填充水平和垂直方向均有效。


    行内转换级块元素：

       display: block;


    级块元素转换成行内元素：

       display: inline;


    转换可变元素：

       display: inline-block;

       float: left|right;



五、CSS (Cascading Style Sheets) 层叠样式表

    http://www.w3school.com.cn/css/css_syntax.asp

    CSS是一种用来表现HTML文档样式的计算机语言。

    CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

    1、CSS样式的作用：

       1）提供更丰富的样式定义；

       2）实现内容（元素）与表现（样式）的分离；样式可以在元素外部定义，元素应用这个样式；

       3）一处定义多处使用，提高开发效率，比如：

              一个样式，多个元素引用；

              一个外部样式表，多个页面应用。

    2、样式表的分类：

       1）外部样式表

          外部的样式文件“***.css”中定义的样式。

       2）内部样式表（也称为文档样式表）

          HTML文档头部<head>标签中的<style>标签中定义的样式。

       3）内联样式表（也称为行内样式表）

          元素的style属性的值中定义的样式。

    3、语法格式：

       CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。

       每条声明由一个属性和一个值组成。

       选择器 {

           属性1: 值1;

           属性2: 值2;

           ......

       }


六、CSS选择器

    要使用css对HTML页面中的元素实现一对一，一对多或者多对一的控制，这就需要用到CSS选择器。

    HTML页面中的元素就是通过CSS选择器进行控制的。

    每一条css样式定义由两部分组成，形式如下： [code] 选择器{样式} [/code] 在{}之前的部分就是“选择器”。
   “选择器”指明了{}中的“样式”的作用对象，也就是“样式”作用于网页中的哪些元素。


    1、ID选择器

    2、类选择器

    3、伪类选择器

       http://www.w3school.com.cn/cssref/css_selectors.asp

    4、标签选择器

    5、复合选择器（又称为后代选择器）

    6、群组选择器

    7、兄弟选择器

    8、属性选择器

    9、通用选择器
    
    
    层叠：

        多个选择器作用于一个元素：
 
            未重名的属性，直接作用于元素；

            重名的属性，若存在优先级，优先级高的属性作用于元素；若优先级相同，按照书写顺序，最后的属性作用于元素。

        多个重名选择器：

            未重名的属性，均作用于元素；

            重名的属性，按照书写顺序，最后的属性作用于元素。注意：不考虑!important最高优先级。


    选择器优先级：

        选择器的权值：

            标签选择器         0 0 0 1

            伪类选择器         0 0 1 0

            类选择器           0 0 1 0

            ID选择器           0 1 0 0

            !important         1 0 0 0

            优先级从低至高。
